### 实例演示：什么是函数？

```markdown
# 什么是函数？

## 什么是函数？

函数是一段可以重复使用的代码块。当你需要多次执行相同的操作时，把代码写成函数，就不用重复写了。

举个例子：函数就像一个榨汁机。你放入水果（输入），它给你果汁（输出）。每次想喝果汁，只需要放入水果，按下按钮就行了。

## 函数的工作原理

函数是如何工作的呢？我们分三步来理解：

### 第一步：定义一个简单函数

先看最简单的函数：

​```python
def say_hello():
    print("Hello!")
​```

这个函数叫`say_hello`，它的功能就是打印"Hello!"。

调用方法很简单：

​```python
say_hello()  # 输出：Hello!
​```

### 第二步：带参数的函数

函数可以接收参数，让它更灵活：

​```python
def say_hello(name):
    print(f"Hello, {name}!")
​```

现在我们可以向不同的人打招呼：

​```python
say_hello("Alice")  # 输出：Hello, Alice!
say_hello("Bob")    # 输出：Hello, Bob!
​```

### 第三步：带返回值的函数

函数还可以返回结果：

​```python
def add(a, b):
    return a + b
​```

使用这个函数：

​```python
result = add(3, 5)
print(result)  # 输出：8
​```

注意到了吗？`return`把结果返回给调用者。

## 常见问题

### 什么时候需要用函数？

当你发现同样的代码写了两遍或更多，就应该考虑用函数。

### 函数和变量有什么区别？

变量是存储数据的，函数是执行操作的。变量像容器，函数像工具。

## 实践练习

现在轮到你了！

**任务**：写一个函数`multiply`，计算两个数的乘积。

**提示**：
- 使用`def`定义函数
- 函数需要两个参数
- 使用`return`返回结果

**参考答案**：

​```python
def multiply(a, b):
    return a * b

print(multiply(4, 5))  # 应该输出：20
​```

## 小结

本文我们学习了：

1. 函数是可重复使用的代码块
2. 函数可以接收参数，让代码更灵活
3. 函数可以返回值，把结果传递给调用者

掌握了函数，你就能写出更简洁、更易维护的代码。

下一节，我们将学习如何使用循环来重复执行任务。
```

## 使用指南

### 如何选择模板

- **讲解新概念** → 使用模板1
- **对比多个方案** → 使用模板2
- **手把手教实现** → 使用模板3
- **解决具体问题** → 使用模板4
- **快速上手工具** → 使用模板5

### 自定义建议

1. 保持模板结构，但根据实际内容调整小节
2. 保持廖雪峰风格的语气和用词
3. 确保每个代码示例都能运行
4. 控制文章长度在2000-5000字
5. 每个章节都有明确的目的

### 质量检查清单

写完文章后，检查：

- [ ] 标题是问题式或动作式
- [ ] 开头引出了问题或场景
- [ ] 每个概念都有通俗解释或类比
- [ ] 代码示例完整且能运行
- [ ] 使用了短句，没有长难句
- [ ] 包含实践练习或验证步骤
- [ ] 结尾有总结和后续引导
- [ ] 语气亲切、鼓励、循序渐进
